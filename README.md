# og-packs

The 40 official CESP sound packs from [peon-ping](https://github.com/PeonPing/peon-ping).

Each pack contains an `openpeon.json` manifest (CESP v1.0) and a `sounds/` directory with audio files.

## Using these packs

Install [peon-ping](https://github.com/PeonPing/peon-ping) and use any pack:

```bash
peon --pack glados
```

Or browse all packs at [openpeon.com/packs](https://openpeon.com/packs).

## Creating your own pack

See [openpeon.com/create](https://openpeon.com/create) for the full guide.
